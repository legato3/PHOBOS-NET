# Contributing to PHOBOS-NET

First of all: thank you for your interest in contributing to PHOBOS-NET.

PHOBOS-NET is a read-only network observability platform focused on clarity, correctness, and calm UX. Contributions are welcome as long as they respect the projectâ€™s core principles.

## Project Philosophy

PHOBOS-NET is:
- Observational, not reactive
- Truth-first (no guessing, no inferred data)
- Calm and explainable
- Designed for long-running, trusted operation

PHOBOS-NET is not:
- An IDS / IPS
- An alert-heavy SOC tool
- An automation or response platform

## Contribution Guidelines

- Truth over completeness
- Observational first
- No silent behavior changes
- Calm UX over alarmist UI

## Submitting a Pull Request

- Keep PRs small and focused
- Explain what changed and why
- Avoid mixing unrelated changes
- Ask first if unsure

Security issues must follow SECURITY.md.

Thank you for helping keep PHOBOS-NET trustworthy.
