# PHOBOS-NET UI Tokens
## Status Card v3 tokens (canonical)

These tokens keep Status Card v3 consistent across the UI. Avoid raw color literals inside the component.

### Color tokens
- `--card-bg`: card fill (glass)
- `--card-border`: thin cyan border
- `--card-border-hover`: hover border
- `--card-glow`: subtle outer glow
- `--card-header-bg`: header wash
- `--card-divider`: divider lines

### Text tokens
- `--text-0`: primary text
- `--text-1`: secondary
- `--text-2`: meta
- `--text-3`: muted labels

### Semantic state tokens
- `--signal-ok`: ACTIVE
- `--signal-warn`: STALE
- `--signal-crit`: OFFLINE

### Typography tokens
- `--font-mono`: mono family
- `--stat-label-size`
- `--stat-value-size`
- `--stat-subvalue-size`
- `--hero-value-size`
- `--hero-unit-size`

### Sizing tokens
- `--status-card-radius`
- `--status-card-padding-x`
- `--status-card-padding-y`
- `--status-card-hero-minh`
- `--status-card-statstrip-minh`
- `--status-card-statstrip-maxh`
- `--status-card-subvalue-minh`
