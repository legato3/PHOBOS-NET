# Security Policy

PHOBOS-NET is a read-only observability platform.

## Reporting a Vulnerability
If you discover a security issue, please open a private issue or contact the maintainer directly.

Do not disclose vulnerabilities publicly before coordination.

## Scope
- Docker image
- Ingestion endpoints (Syslog, NetFlow, SNMP)
- Web UI

This project does not perform active mitigation or enforcement.
